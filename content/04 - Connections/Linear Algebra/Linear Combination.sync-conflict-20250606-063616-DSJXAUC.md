**Idea**: Essentially we can build a set of vectors as a weighted sum of vectors in a given vector space

**Definition**: let $M = \{\vec{v_1},\vec{v_2},.....,\vec{v_m}\}$ be a collection of vectors in a vector space $S$.
- A linear combination of vectors $M$ is a sum of the form:
$$
	\sum_{i=1}^{n} a_i \vec{v_i} = a_1\vec{v_1}+a_2\vec{v_2}+...+a_n\vec{v_n}
$$
- for some $a_1,a_2,...,a_n \in R$
