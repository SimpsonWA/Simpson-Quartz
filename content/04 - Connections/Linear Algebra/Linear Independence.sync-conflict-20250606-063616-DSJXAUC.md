**Idea** Similar to [[Linear Dependence]], linear independence is where we have a finite set of vectors in a vector space and can only obtain a zero vector if the scalar multiples are all 0.

**Definition** A finite set of vectors $\vec{v_1},\vec{v_2},.....,\vec{v_n}$  in a vector space $S$ is linearly independent if 
$$
a_1\vec{v_1} + a_2\vec{v_2} +... + a_n\vec{v_n} = \vec{0}
$$
only when all $a_k = 0$
