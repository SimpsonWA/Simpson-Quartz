**Idea**: Essentially we have a set of vectors in a vector space and we can have a set of scalars that causes the sum of these to be a zero vector without the scalars being 0

**Defeinition** A finite set of vectors $\vec{v_1},\vec{v_2},.....,\vec{v_n}$ in a vector space $S$ is said to be linearly dependent if there exists scalars $a_1,a_2,...,a_n \in R$ not all equal to zero such that:
$$
a_1\vec{v_1} + a_2\vec{v_2} +... + a_n\vec{v_n} = \vec{0}
$$
