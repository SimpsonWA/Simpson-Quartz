**Global Minimizer**:
A point $x^*$ in the domain of the objective function $f(x)$ such that $f(x^*) \leq f(x)$ for all $x$ in the domain. This represents the lowest value of the objective function over the entire domain.

Also if a function $f$ is [[Convex]] then any [[Local Minimizer]] $x^*$ is also a **global minimizer**.
If $f$ is [[Convex]] and differentiable, any **stationary point** $x^*$ is a **global minimizer**.
